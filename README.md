# GF2 Gacha Tracker

少女前線2：追放 抽卡統計工具。

## 功能

- 抽卡紀錄同步
- 定向採購統計
- 軍備提升統計
- 常規採購統計
- UP / 歪統計
- 招募歷史分析
- 最歐 10 連
- 匯入 / 匯出備份

## 使用方式

請至 Releases 下載最新安裝檔。

使用者需自行取得 gachaUrl 與 Authorization 後，於工具內同步設定中填入。

## 注意事項

本工具僅讀取抽卡紀錄 API，不會修改遊戲資料。

請勿分享自己的 Authorization。

【手動匯入歷史資料教學】

由於官方 API 僅能取得近約 6 個月內的抽卡紀錄，因此若您有自行保存較早期的抽卡資料，可透過手動匯入功能補充。

使用方式：

1. 點選「匯入/匯出」→「下載手動模板」
2. 使用記事本或其他文字編輯器開啟模板
3. 依照格式填入歷史抽卡資料
4. 儲存為 JSON 檔案
5. 點選「匯入/匯出」→「匯入手動紀錄」
6. 選擇剛剛編輯完成的 JSON 檔案

範例：

[
{
"time": "2025-01-01 12:00:00",
"source": "定向採購",
"type": "人形",
"name": "黛煙",
"rarity": "橙色"
},
{
"time": "2025-01-02 15:30:00",
"source": "軍備提升",
"type": "武器",
"name": "武器名稱",
"rarity": "紫色"
}
]

欄位說明：

time 抽卡時間（格式：YYYY-MM-DD HH:mm:ss）

source 卡池類型，可填：

* 定向採購
* 軍備提升
* 常規採購

type 類型，可填：

* 人形
* 武器

rarity 稀有度，可填：

* 橙色
* 紫色
* 藍色

注意事項：

* 手動匯入僅建議用於補充官方 API 無法取得的歷史資料。
* 若資料時間落在目前 API 已同步的範圍內，系統將自動跳過，不會覆蓋官方同步資料。
* 格式錯誤或缺少欄位的資料將自動略過。
* 匯入完成後，系統會顯示新增筆數與跳過筆數。
